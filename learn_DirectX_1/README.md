### |      ^     & ~

```c++
// | and
WS_THICKFRAME | WS_MINIMIZE
// ^
0 0 -> 0
1 1 -> 0
0 1 -> 1
1 0 -> 1
WS_OVERLAPPEDWINDOW ^ (WS_THICKFRAME | WS_MAXIMIZEBOX) WS_OVERLAPPEDWINDOWの中に(WS_THICKFRAME | WS_MAXIMIZEBOX)があったら削除、なかったら追加
//　& ~　削除
WS_OVERLAPPEDWINDOW & ~ (WS_THICKFRAME | WS_MINIMIZE and)

```
